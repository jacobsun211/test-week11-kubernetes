git clone https://github.com/yosefanaliza/streamlit_minikube_app.git



then here i installed dependencies:
```
pip install -r requirements.txt
```

minikube:
```
minikube start
```
then did git add, commit, and push
created branch "Dockerfile"
```
git checkout -b Dockerfile
```


build the image:
```
docker build -t jacobsun211/my-repo:latest .
```


run it and see it worked
```
docker run -p 8080:8501 jacobsun211/my-repo:latest 
```


pushing the image:

```
docker login
docker push jacobsun211/my-repo:latest


then did git add, commit, and push
```
git add .
git commit -m "yada yada"
git push --set-upstream origin Dockerfile
```

```
git checkout main
